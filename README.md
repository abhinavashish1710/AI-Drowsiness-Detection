# 😴 AI Drowsiness Detection

A real-time AI-based drowsiness detection system developed using **Python**, **OpenCV**, and **MediaPipe**. The application monitors the user's face through a webcam and detects signs of drowsiness by analyzing facial landmarks.

---

## 🚀 Features

- Real-time webcam monitoring
- Face detection
- Facial landmark detection
- Eye tracking
- AI-based fatigue monitoring
- Simple and lightweight implementation

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

---

## 📦 Installation

```bash
git clone https://github.com/yourusername/AI-Drowsiness-Detection.git
```
